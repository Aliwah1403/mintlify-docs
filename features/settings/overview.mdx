The Settings section is your control center for personalizing Travada and managing workspace configurations. This is where you adjust appearance, control notifications, and manage disabled vehicles and drivers.

<Note>
**🔒 Admin Only** - Most Settings features require Admin privileges. Staff users have limited access to personal preferences only.
</Note>

## What You Can Configure

Settings is organized into two main areas:

### Workspace Settings

Manage team-wide configurations:

- **Appearance** - Switch between light and dark modes
- **Vehicle Management** - View, enable, or delete disabled vehicles
- **Driver Management** - Manage inactive drivers

### Account Settings

Control your personal preferences:

- **Notifications** - Email alerts and system updates
- **Profile** - Your account information
- **Preferences** - Personal workspace settings

## Who Can Access What

| Setting Type | Admin | Staff |
|--------------|-------|-------|
| Appearance | ✅ Full access | ✅ Full access |
| Vehicle Management | ✅ Full access | ❌ No access |
| Driver Management | ✅ Full access | ❌ No access |
| Notifications | ✅ Full access | ✅ Full access |

Staff users can customize their own experience (appearance, notifications) but cannot manage workspace-level settings like vehicles and drivers.

## Why Settings Matter

**For Day-to-Day Comfort:**
- Switch to dark mode if you work late nights
- Turn off notifications for report generation if you generate many reports
- Enable weekly summaries to stay updated

**For Fleet Management:**
- Re-enable vehicles when contracts are renewed
- Permanently delete vehicles that will never return
- Clean up your driver list by removing ex-employees

**For Team Coordination:**
- Ensure everyone gets critical system updates
- Set up report alerts so teams know when invoices are ready
- Configure analytics summaries for performance tracking

## Common Settings Workflows

**New Admin Onboarding:**
1. Set preferred appearance (light/dark)
2. Enable all notification types initially
3. Review disabled vehicles and drivers
4. Adjust notifications based on actual usage

**Seasonal Fleet Changes:**
1. Go to Vehicle Management
2. Re-enable vehicles for busy season
3. Disable vehicles during slow months
4. Keep historical data intact

**End of Month:**
1. Enable report generation alerts
2. Check weekly analytics summaries
3. Review notification settings
4. Adjust for next month's needs

## How to Access Settings

From any page in Travada:

<Steps>
  <Step title="Click Settings icon">
    Look for the gear icon in the main navigation menu
  </Step>
  
  <Step title="Choose your category">
    Select either Workspace (General) or Account (Notifications)
  </Step>
  
  <Step title="Make changes">
    Adjust settings as needed - changes save automatically
  </Step>
</Steps>

## Settings Don't Affect Historical Data

Important to understand:

- Changing appearance doesn't alter any data
- Disabling/enabling vehicles preserves all trip history
- Managing drivers keeps historical trip records
- Notification changes only affect future alerts

All settings are about **how you interact with Travada**, not about the data itself.

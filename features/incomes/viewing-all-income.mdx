## Overview

The **All Income** tab provides a comprehensive view of every revenue source from your trips, combining both Rate fees and Detention fees in one place. This is your go-to view for complete income verification and analysis.

<img src='/images/income-all-tab.png' alt='All Income Tab' />

## What's Displayed

The All tab shows a complete table with:

| Column            | Description                | Example      |
| ----------------- | -------------------------- | ------------ |
| **Registration**  | Vehicle that made the trip | KBE 127H     |
| **Driver**        | Who drove the trip         | John Kamau   |
| **Destination**   | Delivery location          | JAVA CBD     |
| **Trip Date**     | When trip occurred         | Jan 29, 2026 |
| **Rate**          | Base trip charge           | KES 15,000   |
| **Detention Fee** | Container delay charges    | KES 5,000    |

**Total Income:** Displayed at top right (sum of all Rate + Detention in filtered view)

## Accessing the All Tab

<Steps>
  <Step title='Navigate to Income'>
    Click **Income** in the sidebar Or click **Incomes** (depending on your UI)
  </Step>

  <Step title='All Tab'>
    The **All** tab is selected by default Shows all income immediately
  </Step>

  <Step title='Apply Filters (Optional)'>
    Use filters to narrow down the view See filtering section below
  </Step>
</Steps>

## Understanding the Data

### Income Composition

Each row represents one trip with its income breakdown:

**Example 1: Standard Trip**

- Registration: KDA 631Q
- Rate: KES 15,000
- Detention: KES 0
- **Total for this trip:** KES 15,000

**Example 2: Container Trip with Detention**

- Registration: KCG 877T
- Rate: KES 125,000
- Detention: KES 15,000
- **Total for this trip:** KES 140,000

**Example 3: Multiple Trips**
If you have 10 trips:

- Some with just Rate
- Some with Rate + Detention
- **Total Income:** Sum of all Rate + Detention columns

<Info>
  The All tab is particularly useful for seeing which trips generated additional
  detention income beyond the base rate.
</Info>

## Filtering All Income

### Available Filters

<AccordionGroup>
  <Accordion title='Date Range Filter' icon='calendar'>
    **Quick select buttons:** - **7D** - Last 7 days - **1M** - Last 30 days
    (default) - **3M** - Last 90 days - **Custom** - Pick any date range **How
    to use:** 1. Click desired quick button, or 2. Click calendar icon for
    custom range 3. Table updates automatically **Example:** - Want January
    income? Custom: Jan 1 - Jan 31 - Want this week? 7D
  </Accordion>

  <Accordion title='Vehicle Registration' icon='truck'>
    **Filter by specific vehicle(s):** **How to use:** 1. Click **Data Filter**
    button 2. Select **Registration** dropdown 3. Choose one or more vehicles 4.
    Click Apply **Use cases:** - Income generated by KBE 127H - Compare two
    vehicles - Client-dedicated vehicle income
  </Accordion>

  <Accordion title='Destination' icon='location-dot'>
    **Filter by delivery location:** **How to use:** 1. Click **Data Filter** 2.
    Select **Destination** dropdown 3. Choose location(s) 4. Apply **Use
    cases:** - All income from JAVA CBD - Client-specific income (if consistent
    destination) - Route profitability
  </Accordion>

  <Accordion title='Rate Fee Amount' icon='money-bill'>
    **Filter by rate value:** **How to use:** 1. Click **Data Filter** 2. Enter
    rate amount 3. Apply **Use cases:** - Find all KES 15,000 trips - High-value
    trips only - Standard rate verification
  </Accordion>
</AccordionGroup>

### Filter Combinations

Combine multiple filters for specific analysis:

<Tabs>
  <Tab title='Client Billing'>
    **Goal:** Get all income for ARTCAFFE KISUMU in February **Filters:** 1.
    Date: Feb 1 - Feb 28 (custom) 2. Destination: ARTCAFFE KISUMU 3. Apply
    **Result:** Every trip to that client in February **Next:** Generate report
    for invoicing
  </Tab>

  <Tab title='Vehicle Performance'>
    **Goal:** How much did KCG 877T earn this month? **Filters:** 1. Date: 1M
    (last 30 days) 2. Registration: KCG 877T 3. Apply **Result:** Total income
    at top right **Analysis:** Compare to vehicle expenses
  </Tab>

  <Tab title='High-Value Trips'>
    **Goal:** Find all trips over KES 100,000 **Filters:** 1. Date: 3M or 6M 2.
    Rate: 100000 or higher 3. Apply **Result:** Premium trips only **Use:**
    Identify best-paying routes/clients
  </Tab>
</Tabs>

## Understanding Totals

The **Total Income** at top right shows:

**Without filters:**

- Sum of all Rate fees
- Plus sum of all Detention fees
- For entire date range

**With filters:**

- Only includes filtered trips
- Recalculates based on visible rows
- Updates in real-time

**Example:**

- All trips: KES 1,000,000
- Filter to JAVA CBD: KES 150,000
- Total updates to KES 150,000

<Tip>Use filters + Total to quickly answer "How much did we earn from X?"</Tip>

## Common Workflows

### Daily Income Check

<Steps>
  <Step title='Open All Tab'>Income → All (default view)</Step>

  <Step title='Filter to Today'>Set date to today only (custom range)</Step>

  <Step title='Verify Trips'>
    Check all trips are recorded Compare to trip sheets
  </Step>

  <Step title='Note Total'>
    Total Income = today's earnings Should match expectations
  </Step>
</Steps>

### Weekly Review

<Steps>
  <Step title='Set 7D Filter'>Click **7D** button</Step>

  <Step title='Scan for Anomalies'>
    - Any unusually high/low rates? - Missing trips? - Unexpected detention
    fees?
  </Step>

  <Step title='Verify Total'>
    Compare week's total to: - Previous week - Expected volume -
    Budget/projections
  </Step>
</Steps>

### Client Income Summary

<Steps>
  <Step title='Filter by Destination'>Select client's delivery location</Step>

  <Step title='Set Date Range'>Billing period (e.g., full month)</Step>

  <Step title='Review Trips'>
    Verify each trip is legitimate Check for duplicates or errors
  </Step>

  <Step title='Note Total'>Total = amount to invoice client</Step>

  <Step title='Generate Report'>Switch to Rate tab → Generate Report</Step>
</Steps>

## Export Income Data

To download income data for external analysis:

<Steps>
  <Step title='Apply Desired Filters'>
    Set up date range and any other filters
  </Step>

  <Step title='Click Export'>Look for Export button (top right area)</Step>

  <Step title='Choose Format'>
    - Excel (.xlsx) - For spreadsheet analysis - CSV (.csv) - For import to
    accounting
  </Step>

  <Step title='Download'>File downloads with filtered data only</Step>
</Steps>

**What's exported:**

- All visible columns
- Only filtered trips
- Current sort order
- Totals may be included (format dependent)

## Common Questions

<AccordionGroup>
  <Accordion title='Why are some trips showing KES 0?' icon='circle-question'>
    **Rate = 0:** - Trip recorded without rate fee - Possibly return trip or
    deadhead - Or data entry error **Detention = 0:** - Normal! Most trips don't
    have detention - Only container/port trips usually have detention
  </Accordion>

  <Accordion title='Can I edit income here?' icon='pen'>
    **Answer:** No **Reason:** Income is read-only view of trip data **To
    edit:** Go to Expenditure, find trip, delete and re-record See: [Editing
    Trips](/features/expenditure/editing-trip)
  </Accordion>

  <Accordion title="Total doesn't match dashboard" icon='not-equal'>
    **Causes:** 1. Different date filters 2. Dashboard showing different period
    3. Need to refresh **Fix:** - Check date range on both - Refresh browser -
    Ensure same filters
  </Accordion>

  <Accordion title="Why can't I see recent trip?" icon='eye-slash'>
    **Causes:** 1. Date filter too narrow 2. Trip not yet saved (check
    Expenditure) 3. Need page refresh **Fix:** - Expand date range - Refresh
    page (F5) - Verify trip exists in Expenditure
  </Accordion>
</AccordionGroup>

## All vs. Rate vs. Detention

When to use each tab:

<Tabs>
  <Tab title='All Tab'>
    **Use when:** - Need complete income picture - Verifying total revenue -
    Comparing to expenses - General income review - Export for accounting
    **Shows:** Everything
  </Tab>

  <Tab title='Rate Tab'>
    **Use when:** - Generating client invoices - Standard trip billing -
    Rate-only analysis - Most common workflow **Shows:** Only rate fees
  </Tab>

  <Tab title='Detention Tab'>
    **Use when:** - Port/container operations - Detention-specific billing -
    Verifying delay charges - Less frequent use **Shows:** Only detention fees
  </Tab>
</Tabs>

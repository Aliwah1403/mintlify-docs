## Overview

The **Rate** tab shows income from base trip charges - what you invoice clients for transport services. This is typically your largest and most consistent revenue source.

<img src='/images/income-rate-tab.png' alt='Rate Tab' />

## What are Rate Fees?

**Rate fees** are the primary charges for transporting goods from point A to point B. They represent:

- ✅ Base trip cost
- ✅ What you quote clients
- ✅ Agreed-upon amount before trip
- ✅ Most consistent income source
- ✅ Usually 80-95% of total income

<Info>
  Think of rate fees as your standard pricing for transport services, before any
  additional charges like detention.
</Info>

## Rate Tab vs. All Tab

Understanding the difference:

| Feature     | All Tab           | Rate Tab                 |
| ----------- | ----------------- | ------------------------ |
| **Shows**   | Rate + Detention  | Rate only                |
| **Total**   | Combined income   | Rate fees only           |
| **Use For** | Complete overview | Client invoicing         |
| **Reports** | Not available     | Generate Rate Reports ✅ |

<Tip>
  Use **Rate tab** when you need to invoice clients for standard transport
  services.
</Tip>

## What's Displayed

The Rate tab shows:

| Column           | Description                |
| ---------------- | -------------------------- |
| **Registration** | Vehicle that made the trip |
| **Driver**       | Who drove                  |
| **Destination**  | Delivery location          |
| **Trip Date**    | When trip occurred         |
| **Rate Fee**     | Amount charged (KES)       |

**Total Rate Fee:** Sum displayed at top right

<Note>
  Detention fees are **not** shown in this tab - they're separate in the
  Detention tab.
</Note>

## Using the Rate Tab

### Viewing Rate Income

<Steps>
  <Step title='Navigate to Rate Tab'>Income → **Rate** tab</Step>

<Step title='Apply Filters'>Date range, vehicle, destination as needed</Step>

  <Step title='Review Data'>
    - Check all trips recorded - Verify amounts are correct - Note total at top
    right
  </Step>
</Steps>

### Filtering Rate Fees

Same filtering options as All tab:

**Available filters:**

- Date range (7D, 1M, 3M, custom)
- Vehicle registration
- Destination
- Rate amount

## Generating Rate Reports

The Rate tab has a special feature: **Report Generation**

### How to Generate a Rate Report

<Steps>
  <Step title='Filter to Desired Trips'>
    Set date range and destination for the client
  </Step>

<Step title='Click Generate Report'>
  Button located in the rate tab
  <img
    src='/images/generate-rate-report-button.png'
    alt='Generate Report Button'
  />
</Step>

<Step title='Fill Report Details'>
  **Required fields:** - Report Name (e.g., "ARTCAFFE February 2026") -
  Client/Customer Name - VAT Rate (typically 16%) **Optional fields (for
  container trucks):** - Container Number - Weight - Size (20ft/40ft) - Pickup
  Location
  <img src='/images/rate-report-form.png' alt='Rate Report Form' />
</Step>

<Step title='Submit'>Click **Generate** or **Create Report**</Step>

  <Step title='Access Report'>
    Report appears in Reports section Can be viewed, downloaded, shared
  </Step>
</Steps>

See detailed guide: [Generating Rate Reports](/features/reports/generating-rate-report)

### What's Included in Rate Reports

The generated report contains:

✅ **Client information** (name, report title)  
✅ **Trip details** (dates, destinations, vehicles)  
✅ **Rate fees** for each trip  
✅ **Subtotal** (sum of all rates)  
✅ **VAT calculation** (at specified %)  
✅ **Grand Total** (subtotal + VAT)  
✅ **Container details** (if applicable)  
✅ **Professional formatting** (ready to send)

<Info>
  Rate reports are invoice-ready documents you can email directly to clients for
  payment.
</Info>

## Common Questions

<AccordionGroup>
  <Accordion
    title='Why is rate different from what we agreed?'
    icon='hand-holding-dollar'
  >
    **Possible reasons:** 1. Data entry error (typo) 2. Rate changed
    mid-contract 3. Extra charges included in rate field 4. Currency/decimal
    error **Solution:** - Verify with client agreement - Check trip sheet - Edit
    trip if wrong (delete/re-record)
  </Accordion>

<Accordion title='Can I add rate without a trip?' icon='plus'>
  **Answer:** No All rate income must be linked to a trip **If you need to:**
  Record a trip entry even for adjustments
</Accordion>

<Accordion title='What if rate includes detention?' icon='question'>
  **Best practice:** Separate them **Entry:** - Rate: KES 125,000 - Detention:
  KES 15,000 **Why?** Better reporting and analysis **If combined:** Put total
  in Rate, leave Detention blank
</Accordion>

  <Accordion title='How to handle rate discounts?' icon='percent'>
    **Option 1:** Enter discounted amount - Rate: KES 12,000 (after 20%
    discount) **Option 2:** Note in external system - Rate: KES 15,000 (full) -
    Apply discount in accounting software **Travada doesn't track:** Original
    vs. discounted rates
  </Accordion>
</AccordionGroup>
